### トピック1: DeepSeekが旧APIエンドポイントを廃止、V4系への移行が必須に
📅 2026年07月24日公開（出典: DeepSeek / LLM Gateway）

中国のDeepSeekが、従来の**旧APIエンドポイント `deepseek-chat` と `deepseek-reasoner`** の提供を、**日本時間7月25日未明（UTC 7月24日 15:59）に終了**しました。開発者は接続先を **`deepseek-v4-pro` または `deepseek-v4-flash`** へ切り替える必要があります。

エンドポイントとは、プログラムがAIに接続するための「窓口」のこと。**既存コードは接続先名を書き換えないと動作しなくなる**ため、旧エンドポイントを利用中のプロジェクトは早急な移行対応が求められます。

### トピック2: WebフレームワークNuxt 4.5公開、ビルド基盤をVite 8 / Rspack 2へ刷新
📅 2026年07月18日公開（出典: Nuxt Blog）

Vueベースのメタフレームワーク **Nuxtがバージョン4.5を公開**しました。最大の変更は、内部ビルド基盤を **Vite 8** と **Rspack 2（Rsbuildベース）** へ更新した点です。ビルドとは、書いたコードを動く形へ変換する処理のことで、この土台の刷新により**開発時のビルド高速化**が期待できます。

このほか、ページを段階的に配信する**実験的なSSRストリーミングモード**、安定したエラーコード体系、新しい `useLayout` コンポーザブルや名前付きビューを追加。**次期メジャー版Nuxt 5への地ならし**も進みました。なお、Vite 8 / Rspack 2への更新に伴い、カスタムプラグイン・設定は移行ガイドでの確認が必要です。

### 🛠 トピック3: Kubernetes v1.37が機能凍結、正式リリースは8月26日
📅 2026年07月23日公開（出典: Kubernetes Contributors / byteiota）

**本日のインフラ・DevOpsコーナー。** コンテナ管理の定番 **Kubernetesの次期版v1.37が、7月22〜23日にコードフリーズ（搭載機能の確定）** を迎えました。正式リリースは **2026年8月26日** の予定です。

今回のフォーカスは、コンテナ実行基盤 **containerd 2.0対応** と **cgroup v2への移行**で、アップグレード前の互換性確認が必須となります。加えて、**Podを再起動せずにCPU/メモリを変更できるIn-Place Pod Resize** や **Dynamic Resource Allocation（DRA）** といった、クラスタをより動的にする機能が引き続き強化されています。運用担当者は、リリース前の今のうちに環境の互換性を点検しておくのが得策です。
