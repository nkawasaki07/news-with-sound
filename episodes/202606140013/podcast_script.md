### トピック2: Firefox 151 が開発者向け新機能を追加
📅 2026年6月公開（出典: Mozilla）

ブラウザ **Firefox 151** が開発者向け機能を拡充。目玉は **Web Serial API** への対応で、ウェブページから直接シリアルポート経由で**マイコンや周辺機器とやり取り**できるようになりました。これまで主にChrome系に限られていた機能がFirefoxにも広がった形です。

あわせて**ローカルネットワークアクセスを許可制**にする仕組みも導入され、**プライバシーとセキュリティが強化**されています。クロスブラウザでの実装が進むことで、ウェブのみで作れるアプリの幅が広がります。

### 💻 トピック3: Python 3.15 がベータ2で機能凍結、組み込み `sentinel` が登場
📅 2026年6月8日公開（出典: Real Python ／ ベータ2は6月2日リリース）

次期 **Python 3.15** が5月のベータ1で**機能凍結（feature freeze）**を迎え、**6月2日にベータ2**が登場。以降はバグ修正と仕上げに専念する段階に入りました。

確定した主な目玉機能:

- **組み込み `sentinel`**（PEP 661）— 「値が渡されなかったこと」を表す番兵オブジェクトを簡潔に生成。`repr()` が分かりやすく、型注釈にもそのまま使える
- **明示的な遅延インポート**（PEP 810）で起動を高速化
- **`frozendict` 組み込み**（PEP 814）= 変更不可なマッピング
- **UTF-8 をデフォルトエンコーディング化**（PEP 686）
- JITコンパイラが**x86-64 Linuxで8〜9%高速化**

正式版は**今秋**の予定です。

### 💻 トピック4: Meta製のRust製型チェッカー「Pyrefly」が 1.0 に到達
📅 2026年6月8日公開（出典: Real Python）

Metaが開発する**Rust製のPython型チェッカー Pyrefly** が **v1.0** に到達。型チェッカーは、実行前に型の食い違いを検出するツールです。

- 売りは**速度** — **キー入力ごとにチェックを走らせられる**ほど高速
- すでに**Instagramの約2,000万行**のコードベースで標準採用、**PyTorch・NumPy** でも利用
- `pyrefly init` で既存の **mypy / Pyright 設定を変換**でき、ゼロから書き直さずに移行可能

Astralの `ty` と並び、**Rust速度の型チェッカー**が選択肢として存在感を増しています。
