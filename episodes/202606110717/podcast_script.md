### トピック1: Appleの基盤モデルフレームワークが外部AIに開放
📅 2026年06月09日公開（出典: MacRumors / Apple Developer）

### トピック2: AIコーディング時代のWeb指針「Modern Web Guidance」が早期公開
📅 2026年06月公開（出典: web.dev / Chrome for Developers）

Chromeチームが、**AIコーディングエージェント向けのガイダンス集「Modern Web Guidance」**を早期プレビューとして公開しました。**アクセシビリティ・性能・セキュリティ**に優れたモダンなWebを作るための、専門家監修のスキル集です。ブラウザ互換性の目安を示す**Baselineと直接連携**し、エージェントが**古い書き方ではなく現在の標準に沿ったコードを生成**できるよう誘導します。AIにコードを書かせる時代の**品質担保策**として注目されています。
