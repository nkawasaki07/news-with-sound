### トピック1: マイクロソフトのWindows ML 2.0、AIモデルを端末内で賢く分散実行
📅 2026年6月2日公開（出典: Microsoft Build 2026 / Windows News）

マイクロソフトが**Build 2026**で発表した**Windows ML 2.0**は、新しい**ONNXランタイム**を搭載。**ハードウェア適応型パーティショニング**により、推論ワークロードを**NPU・GPU・CPU**へ自動分散します。開発者はチップ構成を意識せず、**端末内で大規模言語モデルを効率実行**できるようになり、クラウドに依存しないローカルAIの実用度が向上しました。

### トピック2: Chrome 149が安定版に、CSSギャップデコレーションに対応
📅 2026年6月2日公開（出典: Chrome for Developers / GIGAZINE）

**Chrome 149**が6月2日に安定版公開。目玉は**CSS Gap Decorations**で、**グリッドやフレックスボックスの要素間のすき間**に線などの装飾を直接スタイルできます。これまでの**回避策（余計な要素の追加）が不要**になり、`column-rule-inset` / `row-rule-inset` などの新プロパティで記述可能。**幅・色・位置のアニメーション**にも対応します（Edgeも149から対応）。

### 💻 トピック3: Go 1.25.11、標準ライブラリのセキュリティ修正をリリース
📅 2026年6月2日公開（出典: The Go Programming Language）

Go言語の最新パッチ版**Go 1.25.11**が6月2日に登場。**`crypto/x509`・`mime`・`net/textproto`** のセキュリティ修正が中心で、コンパイラとランタイムのバグ修正も同梱。新機能こそないものの、**本番運用サービスには重要なアップデート**です。Go利用者は早めの適用が推奨されます。
