### トピック1: グーグル「Gemini Omni Flash」が開発者向けAPIで公開
📅 2026年07月01日公開（出典: Google Cloud Blog）

グーグルが動画生成モデル **Gemini Omni Flash** を**パブリックプレビュー**として開発者向けに公開しました。目玉は**対話型編集（conversational editing）**で、生成した動画に対し「人物を差し替える」「背景を照らし直す」といった指示を**自然言語のマルチターン対話**で重ねられます。**登場人物の一貫性や物理的な整合性を保ったまま**、拡張・修正が可能です。モデル名は `gemini-omni-flash-preview` で、**Google AI Studio・Gemini API・Gemini Enterprise Agent Platform** から利用できます。テキストからの一発生成にとどまらず、**会話で作り込める**点が従来との違いです。

### トピック2: Safari先行版が新CSS関数 `calc-mix()` に対応
📅 2026年07月01日公開（出典: WebKit Blog）

アップルの開発者向け先行版ブラウザ **Safari Technology Preview 247** が公開され、新しいCSS関数 **`calc-mix()`** をサポートしました。`calc-mix()` は**2つの値を指定した割合でブレンド（補間）**できる関数で、サイズや余白などを**なめらかに混ぜ合わせる計算**を、従来より**短く直感的に**記述できます。現時点では**先行版での実装段階**ですが、CSSの数式表現を強化する動きとして、今後の正式版での搭載が注目されます。（対応: macOS Golden Gate / macOS Tahoe）

---

### 💻 トピック3: Python高速ツール「uv」が 0.11.27 に更新
📅 2026年07月06日公開（出典: GitHub / Astral）

今日のプログラミング言語コーナーです。**Rust製の高速Pythonパッケージ／プロジェクト管理ツール「uv」**（Astral製）の **v0.11.27** が公開されました。uvはインストールから依存解決・プロジェクト管理までをまとめて高速にこなす、いわば **「Python版のCargo」** を目指すツールです。今回の主な改善点は次のとおりです。

- **ホイールのメタデータ取得**で、無視できるエラーがあっても**処理を継続**できるように
- **Python本体のダウンロード**に**キャッシュ**を利用して高速化
- **直接再インストール時に `site-packages` 全体の走査を回避**する最適化
- `uv workspace list` で**拡張子なしのシェバンスクリプト**を検出

派手さはないものの、**日々のビルド・インストールの待ち時間を着実に削る**改良が続いています。
