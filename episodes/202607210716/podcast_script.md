### トピック1: アトラシアンがJiraを「AIネイティブ開発」の司令塔に拡張
📅 2026年7月15日公開（出典: Atlassian / DevOps.com）

プロジェクト管理ツール**Jira**で知られるアトラシアンが、Jiraを**AIによるソフトウェア開発の司令塔**へと拡張しました。最大のポイントは、Jiraのチケットを **Claude Code・Cursor・GitHub Copilot** といった**コーディングエージェント（コードを自動生成するAI）に直接割り当てられる**ようになったこと。さらに、これらのツールを**チケットの文脈を読み込んだ状態でワンクリック起動**でき、コピー＆ペーストが不要になります。加えて **Jira組み込みのコーディングエージェント**が、チケットから**レビュー可能なプルリクエストまでを自動生成**。これらは**有料プランに追加費用なし**で含まれ、**Codex対応も予定**されています。

### トピック2: Firefox 153が正式リリース、開発者向け新機能が充実
📅 2026年7月21日公開（出典: Mozilla / MDN）

Mozillaが本日、**Firefox 153**を正式リリースしました。今回は**長期サポート版（ESR）**にあたり、約15か月間アップデートが提供されます。開発者向けの目玉は、開発ツールに追加された**「ローカルモード」**。**ローカルフォルダを独自オリジンとして読み込める**ため、`file://`では動かないAPIのテストに**ローカルサーバーを立てる必要がなくなります**。このほか、**デスクトップ版でPicture-in-Picture API**に対応し、動画の**常時最前面表示をサイト側から制御**可能に。JavaScript側でも、ロケール情報を扱う **`Intl.Locale` info** や、テキストファイルをモジュールとして取り込む **`text` インポート属性**に対応しました。

### 🛠 トピック3: cert-manager v1.21.0 が公開
📅 2026年7月中旬公開（出典: Last Week in Cloud Native / cert-manager）

Kubernetes上で**証明書の発行・更新を自動化**する定番ツール **cert-manager** が **v1.21.0** を公開しました。目玉は **ACME Renewal Information（ARI）対応**。これは**認証局が推奨する更新タイミングを伝える仕組み**で、**更新の集中回避**や**緊急失効への追従**がスムーズになります。また、シークレット管理ツール **Vault** 連携で **AWS IAM認証**が利用可能に。一方で **HelmチャートのRBAC・メトリクス設定に3つの破壊的変更**があり、**アップグレード時は事前確認が必須**です。

### 🛠 トピック4: ゼロトラストの土台 SPIFFE/SPIRE が v1.15.2 をリリース
📅 2026年7月中旬公開（出典: Last Week in Cloud Native / SPIFFE）

ゼロトラストの土台となる**ワークロードの身元証明**を担う **SPIFFE/SPIRE** が **v1.15.2** を公開しました。ワークロードの身元証明とは、人間ではなく**プログラムやサービス同士が「自分が確かにこのサービスだ」と証明し合う仕組み**のこと。今回は、JWT形式の証明書に**使い捨て識別子であるJTIクレームを含めるかどうかを、エントリー単位で設定**できるようになりました。さらに、エージェントのWorkload APIに**呼び出し元ごとのレート制限（実験的）**が加わり、過剰なリクエストからの**保護が強化**されています。
