### トピック1: アマゾン CloudWatch「Coding Agent Insights」を公開
📅 2026年7月20日公開（出典: AWS）

アマゾンのクラウド監視サービス CloudWatch が、**AIコーディングエージェントの利用状況を可視化する新機能「Coding Agent Insights」**を発表しました。**Claude Code・Codex・GitHub Copilot** などのエージェントを一つのコンソールでまとめて監視できます。計測は業界標準の **OpenTelemetry メトリクス**をベースにしており、**追加の計装なし**に Claude apps gateway 経由でテレメトリを収集。**トークン消費量・コスト・開発者アクティビティ**を横断的に把握でき、どのチームでAIツールが成果を出しているかをデータで示せます。

### トピック2: Vue 3.6 リリース候補版が登場、Vapor Mode が完成
📅 2026年7月18日公開（出典: Vue.js公式）

フロントエンドフレームワーク Vue の次期版 **3.6 のリリース候補（RC）**が公開されました。最大の目玉は **Vapor Mode の機能完成**です。**仮想DOMを介さず**、単一ファイルコンポーネントを直接DOM操作へコンパイルすることで、極端なケースで**最大97%高速な描画**、バンドルサイズも**20〜50%削減**を実現します。あわせて中核の **`@vue/reactivity` を alien-signals ベースで全面書き換え**し、大規模なコンポーネントツリーでもスケールしやすくなりました。**既存のVue 3コードとの後方互換性は維持**され、段階的な移行が可能です。

> 補足: 仮想DOMは、画面更新のたびにメモリ上に仮想的なDOMツリーを作って差分計算する仕組み。Vapor Modeはこれを省き、直接DOMを更新することで高速・軽量化します。

### 🔒 トピック3: SharePoint に深刻なRCE脆弱性 CVE-2026-50522、PoC公開後に実悪用
📅 2026年7月22日公開（出典: The Hacker News / Help Net Security）

オンプレミス版 **Microsoft SharePoint** に、深刻な遠隔コード実行（RCE）脆弱性 **CVE-2026-50522** が見つかりました。**7月20日に実証コード（PoC）が公開されると、数時間以内に実際の悪用が観測**されています。危険度は最高クラスの **CVSS 9.8**。原因は**信頼できないデータのデシリアライゼーション**で、**認証なしにネットワーク経由で任意コードを実行**できてしまいます。さらに攻撃者は**マシンキー（暗号鍵）を窃取**しており、**パッチ適用後も居座られる恐れ**があります。対策は**パッチ適用に加えてマシンキーのローテーション（鍵の入れ替え）が必須**です。

> 補足: デシリアライゼーションは、送られてきたデータをプログラム内部のオブジェクトに復元する処理。ここに細工されたデータを渡されると任意コードが実行されます。
