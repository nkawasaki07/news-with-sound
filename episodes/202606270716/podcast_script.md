### トピック1: クアイショウが動画生成AI「Kling 3.0」シリーズを公開
📅 2026年6月20日公開（出典: Atlas Cloud / fal）

中国のクアイショウが動画生成AIモデル **Kling 3.0シリーズ** を公開しました（高速版Turboは6月17日に先行）。**最大15秒・1080p** の動画を一度に **最大6カット** まで生成でき、英・中・日・韓・西の **5言語でネイティブのリップシンク音声** も同時生成します。プロンプトを順を追って解釈する **Visual Chain-of-Thought（vCoT）推論** で複雑な指示の理解が向上。**APIでモデルID指定で組み込み可能** で、720pは約 **$0.11/秒**（1080pは約$0.14/秒、いずれも音声込み）と開発者が統合しやすい料金です。

### トピック2: Next.js 16.3が登場、画面遷移を一気に高速化
📅 2026年6月25〜26日公開（出典: Next.js Blog / Vercel）

Webフレームワーク **Next.js 16.3** が公開されました。目玉は **Instant Navigations** — SPA並みの応答性をもたらす新機能群で、画面遷移を **ストリーム / キャッシュ / ブロック** と細かく制御できます。あわせて **エージェント開発向け強化** として、指示をまとめた **AGENTS.md の同梱**、すぐ使える **ファーストパーティ skills**、**Reactを内省できる agent browser** を追加しました。

### 🛠 トピック3: サービスメッシュ「Linkerd 2.20」リリース
📅 2026年6月23日公開（出典: Cloud Native Now / Buoyant）

**今日のインフラ・DevOpsコーナー。** Kubernetes向けサービスメッシュ **Linkerd 2.20** が公開。最大の改善は **コントロールプレーンのメモリ使用量を約85%削減**（destination・identity・proxy-injectorが対象）。さらに **レートリミット対応の負荷分散** で過負荷の上流を回避、**コンテナ外のWindows VM** をメッシュに正式対応、**mTLSのトラストアンカー自動ローテーション** も内蔵しました。
