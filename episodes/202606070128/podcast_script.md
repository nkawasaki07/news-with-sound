### トピック1: GoogleのエージェントSDK「ADK」が正式版に到達、Gemini APIにマネージドエージェント

Googleが、AIエージェント開発キット**ADK（Agent Development Kit）をバージョン1.0として正式公開**しました。**Python・TypeScript・Java・Go**の4言語に対応し、安定性の保証が付きます。あわせて**Gemini API**には、Googleがホストする**安全に隔離されたLinuxサンドボックス**内で自律的・ステートフルに動くエージェントを構築・実行できる**マネージドエージェント機能**がパブリックプレビューで追加されました。開発者は実行環境の管理から解放され、エージェントのロジック設計に集中できます。

（出典: Google 開発者ブログ ほか）

### トピック2: ビルドツール **Vite 8** が登場、Rust製バンドラ「Rolldown」を全面採用

Web開発の土台となるビルドツール**Viteのバージョン8が3月12日に正式リリース**されました。最大の変更点は、内部バンドラを従来の esbuild + Rollup から、**Rust製の Rolldown に完全置換**したこと。**Rollup比で10〜30倍高速**とされ、巨大モノレポでも開発サーバーが瞬時に起動します。**Astro** などのフレームワークが早期検証に協力し、互換性問題を解消してきました。フロントエンドのビルド体験を大きく変える一手です。

（出典: Vite 公式ブログ ほか）

### 💻 トピック3: **Java 26** が正式リリース、AI向け機能を強化

**今日のプログラミング言語コーナーです。** プログラミング言語Javaの最新版、**Java 26が3月17日に正式リリース**されました。LTS（長期サポート）ではない通常リリースで、**合計10件のJEP**を搭載。注目は**AI開発を意識した機能群**で、数値計算を高速化する**ベクターAPI**、並行処理を書きやすくする**構造化並行性**、初期化を遅延させる**遅延定数**などが含まれます。さらに**G1 GCの改善**により、参照書き換えが多い処理で**最大15%のスループット向上**が見込まれます。※一部機能はまだプレビュー段階である点に留意。

（出典: Oracle Java ブログ / InfoQ ほか）
