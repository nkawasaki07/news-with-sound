### トピック1: エヌビディアが完全オープンな大規模モデル「Nemotron 3 Ultra」を公開
📅 2026年06月04日公開（出典: NVIDIA / Artificial Analysis）

エヌビディアが、**重みを完全公開**した大規模言語モデル **Nemotron 3 Ultra** をリリースしました。総パラメータ **5500億**・アクティブ約 **550億** の **MoE（専門家混合）** 構成で、状態空間モデルの **Mamba とアテンションを組み合わせたハイブリッド設計**。**長時間動くAIエージェント**向けに最適化され、ツール呼び出し・サブエージェントへの委任・エラーからの復帰を**数百ターン**にわたって実行できます。**推論は約5倍速く**、複雑なエージェント処理のコストを**最大3割削減**。重みに加え**学習データやレシピもオープンライセンス**で公開されました。

### 💻 トピック2: 5月にWebプラットフォームへ加わった新機能まとめ
📅 2026年06月公開（出典: web.dev「New to the web platform in May」）

web.dev が**5月にブラウザへ追加された新機能**をまとめました。

- **Safari 26.5**: `:open` 擬似クラスに対応。`<details>` `<dialog>` `<select>` などが**開いている状態をCSSで直接スタイリング**可能に。
- **Chrome 148**: `<video>` `<audio>` に **`loading="lazy"`（遅延読み込み）** を導入。画面に近づくまで読み込みを遅らせ**表示を高速化・通信量も節約**。あわせて**名前だけで参照する name-only コンテナクエリが Baseline 入り**。
- **Firefox 151**: **Document Picture-in-Picture API** に対応。**任意のHTMLを常に最前面の小窓**に表示できる。

### 🛠 トピック3: Microsoft Defender for Cloud、サーバーレス保護がGA
📅 2026年06月01日公開（出典: Microsoft Learn）

マイクロソフトのクラウドセキュリティ基盤 **Microsoft Defender for Cloud** で、**サーバーレス保護が正式提供（GA）** になりました。**Azure Functions / Azure Web Apps / AWS Lambda** を自動検出し、**設定ミス・脆弱性・安全でない依存関係**を**クラウドをまたいで評価**します。サーバーレスは管理するサーバーがない分**設定の見落としがリスク**になりがちですが、それを**自動チェック**できるのが利点。あわせて **AWS RDS 向けのオープンソースDB保護も GA** となりました。
