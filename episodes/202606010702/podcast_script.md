### トピック1: OpenAIがセキュアMCPトンネルを公開、社内サーバーとAIを直結

OpenAIが、ChatGPT・**Codex**・**Responses API**・**AgentKit**から、社内やオンプレミスの**MCPサーバー**へ安全に接続できる「**セキュアMCPトンネル**」を企業向けに提供開始しました。MCPはAIに外部データやツールを使わせるための共通接続規格です。**外部に出せない社内データを保ったまま**業務システムをAIに操作させられる点が大きく、エージェント活用の現実的な土台になります。（出典: OpenAI API Changelog）

### トピック3: CSSの新機能が出そろう、JS無しで実現できる表現が拡大

これまで**JavaScriptに頼っていた表現**が、CSS標準だけで書けるようになってきました。

- **@starting-style** が主要ブラウザに到達 — `display: none` からの**入場アニメーション**がJSのタイミング調整なしで実現
- **scroll() / view()** によるスクロール連動アニメーションが**クロスブラウザでBaseline**に
- **Chrome 144** が `::search-text` 疑似要素を追加し、検索ヒット文字の装飾が可能に

JavaScript依存を減らす流れが鮮明です。（出典: LogRocket, web.dev Baseline ほか）
