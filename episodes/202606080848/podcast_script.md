### トピック1: マイクロソフト、自社製コーディングAI「MAI-Code-1-Flash」を発表
📅 2026年6月2日公開（出典: CNBC / Microsoft Build 2026）

開発者会議**Build 2026**で、マイクロソフトが**自社開発のコーディング特化モデル「MAI-Code-1-Flash」**を発表しました。自然言語の指示からアプリやWebサイトの**ソースコードを生成**し、**GitHub CopilotとVS Codeに順次展開**されます。

同時に初の推論モデル**「MAI-Thinking-1」**も公開。**外部モデルからの蒸留に頼らず**、ライセンスの明確なデータでゼロから訓練した点が特徴で、**低トークンコストでの高効率**を打ち出しています。**OpenAIへの依存低減とコスト削減**が狙いです。

### トピック2: Angular 22が正式リリース、「シグナルファースト」時代へ
📅 2026年6月3日公開（出典: Angular公式 / Medium ほか）

Webフレームワーク**Angularがv22を正式リリース**。最大の変化は**シグナルファーストへの全面移行**です。シグナル（値の変化を自動追跡しUIへ反映する仕組み）を中心に、**Signal Forms・Resource API・Angular Ariaが安定版**になりました。

さらに注目は**セレクターレスコンポーネント**。従来は部品ごとに文字列セレクターの命名が必須でしたが、**コンポーネントのクラスをテンプレートへ直接インポート可能**になり、リファクタリング性が大きく向上します。

### 🔒 トピック3: シスコ Catalyst SD-WAN Managerにゼロデイ、root権限で悪用
📅 2026年6月5日公開（出典: Help Net Security / The Hacker News）

**今日のセキュリティコーナー（1件目）。** シスコの**Catalyst SD-WAN Manager**に深刻なゼロデイ脆弱性**CVE-2026-20245**が判明し、**すでに実環境で悪用**されています。

原因はCLIの**入力検証不足**。認証済みのローカル攻撃者が細工ファイルをアップロードすると、**最高権限のrootで任意コマンドを実行**できます。**6月5日時点でパッチも回避策も未提供**で、エッジ機器への設定変更が押し込まれた事例も確認されています。該当製品の利用者は要警戒です。

### 🔒 トピック4: Windows Netlogonに致命的RCE、CVSS 9.8で悪用進行中
📅 2026年6月1日公開（出典: Help Net Security / SecurityWeek）

**セキュリティコーナー（2件目）。** Windowsのドメイン認証を担う**Netlogon**に、深刻なRCE脆弱性**CVE-2026-41089**が見つかり、**実際に悪用**されています。深刻度**CVSSは9.8**と極めて高い水準です。

正体は**スタックベースのバッファオーバーフロー**。ドメインコントローラーとして動作するWindowsサーバーに**細工したネットワーク要求**を送ることで、**遠隔コード実行**が可能になります。マイクロソフトは**先週のPatch Tuesdayで修正済み**のため、サーバー管理者は速やかな適用を。
