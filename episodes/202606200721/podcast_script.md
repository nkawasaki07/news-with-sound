### トピック1: グーグルがGemini CLIの個人向け提供を終了、Antigravity CLIへ移行
📅 2026年6月18日公開（出典: 開発者向け比較記事・各種テックブログ）

グーグルは、ターミナルで動く**AIコーディングエージェント「Gemini CLI」の個人ティア向けリクエスト受付を6月18日に終了**しました。これまで無料で利用できたため多くの開発者が使っていましたが、今後は**Google I/O 2026で発表された「Antigravity CLI」へ置き換え**られます。**エンタープライズ向けのアクセスは継続**。手元のターミナルからAIに開発作業を任せる「エージェント型コーディング」の選択肢が、また一つ世代交代を迎えました。

### トピック2: Tailwind CSSがv4.3.1を公開、CLIに`--silent`オプション追加
📅 2026年6月12日公開（出典: GitHub / tailwindlabs/tailwindcss）

ユーティリティファーストのCSSフレームワーク**Tailwind CSSがv4.3.1をリリース**。`@tailwindcss/cli`に**出力を抑制する`--silent`オプション**が追加され、CI/自動化パイプラインでのログがすっきりしました。あわせて各種バグ修正も含まれます。**v4系はRust製エンジン採用**でビルドが高速化されており、Biomeなどフォーマッタとの連携改善も継続しています。

### 🛠 トピック3: Cilium v1.19.5を公開 — eBPFベースのKubernetesネットワーキング
📅 2026年6月16日公開（出典: GitHub / cilium/cilium releases）

今日のインフラ・DevOpsコーナー。Kubernetes向けCNI**「Cilium」がv1.19.5を公開**しました。Ciliumは**Linuxカーネル内で安全にコードを動かすeBPF**を土台に、**kube-proxyを置き換え**、L3/L4/L7のネットワークポリシーを適用、**サイドカー無しでmTLS暗号化**を実現し、**Hubble**で通信を可観測化します。今回は**1.19系の最新パッチリリース**で、本番クラスターの安定運用を重視するチーム向けの更新です。
