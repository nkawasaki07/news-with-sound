### トピック1: Google NotebookLM が大刷新、Gemini 3.5 とエージェント機能を搭載
📅 2026年6月8日公開（出典: 9to5Google／SiliconANGLE）

Google がリサーチ支援ツール **NotebookLM** を大幅アップデート。新モデル **Gemini 3.5** を採用し、エージェント型コーディング基盤 **Antigravity** を統合しました。これまでの「資料を要約する受け身のツール」から、**自分でコードを書き、隔離された実行環境でデータ分析まで行う能動的なリサーチパートナー**へ進化しています。

- **100以上のソフトウェアスキル**を Antigravity 経由で備え、システムレベルの操作を実行
- 許可すれば**ライブWebを自律的にクロール**して一次資料を発見
- 出力は **PDF・Word・Markdown・CSV・Excel・PowerPoint・チャートなど11形式**に対応
- 推論ステップがチャット上で可視化され、回答の透明性が向上

当面は **Google AI Ultra 契約者**と一部の Workspace 法人顧客向けの提供です。

### トピック2: Chrome 150 ベータ登場、CSSからシステムの色を直接利用可能に
📅 2026年6月上旬公開（出典: Chrome for Developers）

Chrome 149 が6月2日に安定版となったのに続き、次期 **Chrome 150 のベータ**が公開されました。目玉は新システムカラー **`AccentColor` / `AccentColorText`** です。

- ユーザー端末で設定された**OSのテーマ色（アクセントカラー）をCSSから直接参照**できる
- ボタンやリンクをOS設定色に自動でそろえ、**ネイティブアプリのような統一感**を少ないコードで実現
- Chrome 150 では **IndexedDB の SQLite ストレージバックエンド**も搭載予定で、ブラウザ内DBの安定性が向上

安定版リリースは **6月30日**の見込みです。ブラウザがこれまでJavaScriptやライブラリで補っていた機能を、次々とCSS/プラットフォーム側へ取り込む流れが続いています。

### 💻 トピック3: Deno 2.8系、Node.js互換が大きく前進（今日のプログラミング言語コーナー）
📅 2026年6月3日公開（最新パッチ v2.8.2、出典: Deno公式／GitHub Releases）

JavaScript/TypeScript ランタイム **Deno** の **2.8系**が話題です（最新パッチ **2.8.2** が6月3日公開）。「**過去最大のマイナーリリース**」とうたわれ、最大の焦点は **Node.js 互換性**でした。

- Node公式テストの通過率が **約42% → 76.4%（4,457中3,405件）**へ大幅向上
- **`deno install` が npm / yarn / pnpm の置き換え**として機能。`package.json` を読み、`node_modules` 互換レイアウトを生成し、**コールドキャッシュで2.7比3.66倍高速**
- 脆弱なnpmパッケージを自動修正する **`deno audit fix`**、CI向け **`deno ci`**、npm公開用 **`deno pack`** を追加
- **`import defer`**（エクスポート参照時まで評価を遅延）や **TypeScript 6.0.3**、catalog workspaces、CPUフレームグラフにも対応

※補足: 2.8系の主要機能は2.8リリース（5月下旬）で導入され、7日以内に公開された最新版は6月3日のパッチ2.8.2です。今週時点で鮮度・技術性ともに最も適した言語トピックとして採用しました。
