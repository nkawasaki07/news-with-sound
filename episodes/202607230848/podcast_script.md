### トピック1: Claude開発者プラットフォームがエージェントメモリAPIを更新
📅 2026年7月22日公開（出典: Anthropic / Releasebot）

Anthropicの**Claude Developer Platform**が、エージェントメモリ機能の新ベータヘッダー **`agent-memory-2026-07-22`** を公開。

- メモリ一覧の取得結果が**サーバー定義の安定した順序**で返るように変更
- **depth（深さ）・path_prefix（パス絞り込み）・cursor（ページ送り）** の挙動をより厳密化
- 各SDKは新しいメモリストアヘッダーを**デフォルトで送信**するよう更新

エージェントメモリは、AIエージェントが過去のやり取りや作業内容を保存する仕組み。今回の更新で**メモリの扱いが予測しやすく**なり、エージェント開発の信頼性が高まる。

### トピック2: TanStack Start / Router が7月に相次いでアップデート
📅 2026年7月16〜19日公開（出典: TanStack GitHub / This Week In React）

型安全なReact向けツール群 **TanStack** が、Router と Start を立て続けに更新。

- **React Server Components（RSC）対応**を進める `@tanstack/react-start-rsc` を更新、**Vue / Solid 向け Start** パッケージも同時整備
- 開発中の書き換えでも状態を保つ **HMR（ホットモジュールリロード）の改善**
- `decodeSegment` で**URLの特殊文字（percent-encoded）を正しく保持**する修正、`start-client-core` の**ゼロコピーなペイロード抽出**

RSCはサーバー側で描画しブラウザへ送るデータを削減する仕組み。**日々の開発体験を地道に快適にする**パッチ群がそろった。

### 📱 トピック3: Android Developer ID Status APIがグローバル提供へ
📅 2026年7月公開（出典: Android Developers Blog）

Googleが7月、**Android Developer ID Status API** を全世界向けに提供開始し、**Android Developer Console API** の早期アクセスも開始。

- アプリ登録を**一括（バルク）**、または **CI/CDパイプラインから直接**実行できる
- 学生・趣味の開発者向けに、**政府発行IDや登録料なしで最大20台**に配布できる**限定配布アカウント**を用意
- Developer Console APIと限定配布アカウントの**本格グローバル展開は8月予定**

アプリ配布の敷居を下げつつ、**登録作業の自動化**を後押しする開発者向け機能拡充。
