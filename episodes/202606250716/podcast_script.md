### トピック1: Anthropicが「Claude Tag」を公開 — Slackの中で働くAIの同僚
📅 2026年6月23日公開（出典: Fortune / SiliconANGLE / TechRepublic）

Anthropicが**Slack上で動くAIエージェント「Claude Tag」**を発表しました。チャンネルで `@Claude` とタグ付けして呼び出すと、エージェントが**タスクを段階に分解し、承認された会話・ツール・データ・コードベースを参照しながら自律的に作業を完遂**します。

注目ポイントは次の3つです。

- **共有アイデンティティ**: 会社全体で単一の「Claude」を共有でき、メンバー間でやりかけのタスクを引き継げる
- **アンビエント動作**: 忘れられたスレッドやタスクに**自分から follow-up** する能動的な振る舞い
- **スコープ制御**: ツールやデータへのアクセスを限定できる、エンタープライズ前提の設計

現在は**Claude Enterprise / Team 向けのベータ**。Anthropic社内では、製品チームのコード変更の**約65%がClaude Tag経由で取り込まれている**とのことです。

### トピック2: ビルドツール「Vite 8.1」が正式リリース — 巨大アプリの起動が最大15倍速く
📅 2026年6月23日公開（出典: Vite公式ブログ）

Webのビルドツール **Vite 8.1** が正式リリースされました。Vite 8系は3月から**Rust製バンドラー「Rolldown」を全面採用**しており、週間ダウンロードは**4,160万回**に到達しています。

今回の目玉は、開発時の体感速度を上げる実験的な**バンドルド開発モード（bundled dev mode）**です。

- 10,000個のReactコンポーネントを読み込むアプリのテストで、**起動が約15倍・フルリロードが約10倍高速化**
- 実アプリでも、Linearチームが**コールドスタート最大3倍速・リロード約40%速・ネットワークリクエスト1/10**を報告

開発サーバーの「重さ」を解消する一手として、大規模フロントエンドに効きそうな改善です。

### 📱 トピック3: Android XR × Geospatial API × Gemini で作る現実拡張の観光ガイド
📅 2026年6月17日公開（出典: Android Developers Blog）

今日のモバイル・デスクトップコーナー。Googleが、**Android XR** を使った空間コンピューティングの実装事例を公開しました。今回プレビュー提供された **Geospatial API**（ARCore for Jetpack XR）は、Googleの**VPS（視覚測位システム）**で、デジタルコンテンツを現実世界に**サブメートル精度**でアンカーできる技術です。

デモ「XR Geospatial Tour」は、4つの技術を組み合わせています。

- **Geospatial API / VPS**: GPSより高精度な緯度・経度・方位を取得し、3Dウェイポイントを現実に整合
- **Gemini API（Firebase AI Logic）+ Google Maps Grounding**: 周辺ツアーをJSONで自動生成。**座標のハルシネーションをMaps情報で裏付け**して防止
- **gemini-2.5-flash-tts**: テキストではなく**音声を直接返す**ナレーション生成
- **Jetpack XR SDK / Compose for XR**: `SpatialBox` や `SceneCoreEntity` で3Dオブジェクトと2D Compose UIを空間に共存

ハンズフリーの「街歩きガイド」が、Android開発者の手の届く範囲に来たことを示す好例です。**XREAL Project Aura** を含む開発キット提供プログラムへの応募も始まっています。
